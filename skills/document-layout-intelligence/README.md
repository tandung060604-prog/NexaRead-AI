# Document Layout Intelligence

Repository-local guidance for source-faithful document normalization and
semantic layout work. Coding agents must read `SKILL.md` before changing
parsers, processing, layout persistence, reader layout metadata, or any logic
that can affect citation anchors.

- `examples/source-display-contract.json` shows the additive source/display
  contract.
- `evaluation/layout-cases.json` defines the minimum deterministic evaluation
  surface and must grow with behavior changes.
